# Umkhaya / Kumusha Website

A simple and elegant single-page website for **Umkhaya / Kumusha**, a Zimbabwean and Southern African inspired home kitchen business based in Berlin.

## About

This website was created for a small home-based food business that offers:

* Weekend homemade meals
* WhatsApp pre-orders
* PayPal payments
* Small event catering
* Zimbabwean & Southern African inspired dishes

Tagline:

> *A taste of home nourished with wild flavors*

Secondary tagline:

> *A taste of home in every plate*

---

## Features

* Responsive mobile-friendly design
* WhatsApp ordering integration
* Zimbabwean flag-inspired color palette
* Catering section
* Pickup information
* Simple single-file HTML structure
* Beginner-friendly codebase

* HTML5

## How To Run Locally

1. Clone the repository:

```bash
git clone <your-repo-url>
```

2. Open the project folder in VS Code.

3. Open `index.html` in your browser.

OR

Use the VS Code Live Server extension:

* Right click `index.html`
* Select **Open with Live Server**
